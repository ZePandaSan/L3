## Compiler fichier .tex

#### vers pdf
``` 
pdftex cours.tex
```
#### vers html
``` 
latex2html -local_icons -split=3 rapport.tex
``` 

### Windows :
#### vers PDF et html 
Utiliser wsl et se réferer à Linux
ou 
installer MiKTex ou TeXmaker
