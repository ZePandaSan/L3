# L3
Cours de L3 info

## Compiler fichier .tex 
### Linux : 
#### vers pdf
``` 
pdftex cours.tex
```
#### vers html
``` 
latex2html -local_icons -split=3 cours.tex
``` 

### Windows :
#### vers PDF et html 
Utiliser wsl et se réferer à Linux
ou 
installer MiKTex ou TeXmaker
